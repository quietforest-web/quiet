Détails concernant le jeu

Santa Gift Shooter is a casual arcade cannon shooter with Christmas theme. Let's help Santa to shoot all the gifts! Use candy cannon to shoot all of the gifts on each level. Each candy costs money so you'll need to shoot all of the gifts on a level before you run out of money. Shoot the bonus items to help you get more money or powerful shots. Have fun and enjoy Santa Gift Shooter game here at Y8.com!
